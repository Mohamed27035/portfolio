# Portfolio de Mohamed Salem

## Contenu
- `index.html` : page principale
- `styles.css` : design
- `script.js` : menu mobile et année automatique
- `assets/profile.jpg` : photo
- `assets/CV_Mohamed_Salem.pdf` : CV

## Publication rapide avec GitHub Pages
1. Créez un dépôt GitHub nommé `portfolio`.
2. Importez tous les fichiers du dossier.
3. Ouvrez `Settings` > `Pages`.
4. Dans `Build and deployment`, choisissez `Deploy from a branch`.
5. Sélectionnez la branche `main` et le dossier `/root`.
6. Enregistrez.

Le site sera disponible à une adresse proche de :
`https://Mohamed27035.github.io/portfolio/`

## Modification
Ouvrez `index.html` avec Visual Studio Code pour modifier les textes et liens.
